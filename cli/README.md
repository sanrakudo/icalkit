# iCal Tools CLI

Command-line utilities for working with iCalendar files.

## Coming Soon

The CLI version is currently under development. Stay tuned!

## Planned Features

- `ical-split` - Split large iCal files into smaller chunks
- `ical-view` - View and analyze iCal file contents
- `ical-merge` - Combine multiple iCal files
- `ical-clean` - Remove duplicates and clean up calendars

## Installation (When Available)

```bash
npm install -g ical-tools-cli
```
